Data Collection Summary

This app does not collect or share any personal data. All data (calculations, vault files, settings) is stored locally on your device with encryption.

Data Types

Data Type Collected? Stored? Purpose
Personal Info No No —
Contacts No No —
Location No No —
Photos/Videos No Yes (Encrypted, Local) Vault storage — files you choose to hide
Documents No Yes (Encrypted, Local) Vault storage — files you choose to hide
App Activity No Yes (Local only) Calculation history on device
Device ID Yes (AdMob) No Ad delivery only

How Your Data Is Protected

Protection Details
Vault Encryption AES-256 encryption for all hidden files
PIN Storage SHA-256 hash — actual PIN is never stored
Storage Location App-private directory (not accessible by other apps)
Network Transmission None — all data stays on device
Cloud Storage None — no cloud sync or backup
User Accounts None required

Data Sharing

Third Party Data Shared Purpose
Google AdMob Device ID, IP, ad interactions Ad delivery and measurement
Analytics None —

No vault data, file information, PIN data, or content metadata is ever shared with Google AdMob or any third party.

Data Deletion

You can delete your data in the following ways:

  • Unhide files: Decrypt and restore to your device
  • Delete permanently: Remove files from vault forever
  • Clear history: Delete calculation history
  • Uninstall App: Removes all local data

Children's Data

This app does not knowingly collect data from children under 13.
