Data Security

  • All calculation data is stored locally using Hive (local database)
  • No data is transmitted to external servers (except AdMob ad requests)
  • No user accounts or registration required
  • No cloud sync or backup features

Data Deletion

You can delete your calculation history at any time from within the App by using the "Clear History" option. Uninstalling the App will also delete all locally stored data.

Children's Data

This app does not knowingly collect data from children under 13. The app does not contain content directed at children.
